## Installation
> bundle add speedify

## Example usage:
> puts Speedify.speed_to_mps(10)           # => 2.77778 (approx)

> puts Speedify.time_for_distance(42.195, 10) # => "04:13:12"

> puts Speedify.distance_for_time(2, 10)      # => 20.0

> puts Speedify.speed_for_distance_and_time(42.195, 4.2167) # => ~10.0
